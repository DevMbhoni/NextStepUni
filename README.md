## 🚀 About The Project

NextStepUni is a **final-year group project (3 members)** designed to simplify how students explore universities and scholarship opportunities across South Africa.

The application provides a structured, user-friendly desktop experience built using **Java and JavaFX**, with a strong focus on object-oriented design and modular architecture.

---

## ✨ Core Features

🔐 User Authentication (Login / Register)  
🏫 Browse & search universities  
🎓 Filter scholarships  
⭐ Student review system  
🛠 Admin moderation tools  
📊 Structured backend logic using OOP principles
